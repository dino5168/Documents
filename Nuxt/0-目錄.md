# Nuxt 目錄

### 如何建立 Nuxt 專案。

```sh

npm create nuxt <project-name>
```

```sh
git init
git remote add origin https://github.com/dino5168/XA.git
git remote -v
git add .
git commit -m "初始化專案"
git push -u origin main
git add .
git commit -m "更新內容"
git push


```

安裝 Tailwind css

```sh
npm install tailwindcss @tailwindcss/vite
```

設定 tailwindcss

```ts
// https://nuxt.com/docs/api/configuration/nuxt-config
import tailwindcss from "@tailwindcss/vite"; //import tailwindcss Nuxt 3 預設使用 vite
export default defineNuxtConfig({
  compatibilityDate: "2024-11-01",
  devtools: {enabled: true},
  vite: {
    plugins: [tailwindcss()],
    server: {
      hmr: true,
    },
  },
  css: ["~/assets/css/main.css"], // 加入這行 建立目錄 ~/assets/css/main.css
  modules: [
    "@nuxt/content",
    "@nuxt/eslint",
    "@nuxt/fonts",
    "@nuxt/icon",
    "@nuxt/image",
    "@nuxt/ui",
    "@nuxt/scripts",
  ],
});
```
